# La Comarca — Gastropub (Página de presentación)

Esta es la página de presentación de **La Comarca** (Cruz del Eje, Córdoba), pensada para que cualquier persona pueda entrar, entender rápido qué ofrecen y tener a mano la ubicación / cómo contactarlos.

## Qué vas a ver en la página

1. **Inicio (Portada / Hero)**
   - Muestra el estilo del lugar y un botón para ir directo a la sección de ubicación/WhatsApp.

2. **El Lugar (Propuesta)**
   - Explica la idea del local y lo que lo hace especial (comida, tragos y ambiente).

3. **Menú (Nuestras especialidades)**
   - Lista lo principal que se ofrece: hamburguesas, pizzas y cócteles, con una breve descripción.

4. **Ambiente**
   - Un “mosaico” con fotos del lugar.
   - El texto va arriba de la imagen para que se lea bien.

5. **Reseñas**
   - Muestra valoración y comentarios (estilo “lo que dicen”).

6. **Ubicación / Contacto (CTA)**
   - Dirección, horarios y botones para:
     - ver el lugar en Google Maps
     - escribir por WhatsApp
